Getting Started

Create a new database by providing a name and uploading your first CSV file. You can add more tables later from the database management page.

Requirements:

  • • CSV files only (max 50MB per file)
  • • Maximum 5 databases per user
  • • Maximum 10 tables per database
  • • Database names must be unique and URL-friendly

Will be used in your website URL: {{ actor.username }}-[database-name].datasette-portal.fly.dev

What happens next:

  1. 1 Your database will be created in Draft mode
  2. 2 CSV data will be imported as your first table
  3. 3 You can add more tables and customize your site
  4. 4 Publish when ready to make it publicly available

CSV Format Guidelines

  • • First row should contain column headers
  • • Use UTF-8 encoding for special characters
  • • Avoid empty rows or columns
  • • Date format: YYYY-MM-DD recommended

Database Naming Tips

  • • Use descriptive, meaningful names
  • • Lowercase letters and hyphens work best
  • • Avoid spaces and special characters
  • • Keep it concise but clear
