New Workflow

Create an empty database first, then add tables by uploading CSV files. This gives you more control over your data organization.

Requirements:

  • • Database names must be globally unique
  • • Use lowercase letters, numbers, hyphens, and underscores
  • • Maximum 5 databases per user
  • • Maximum 10 tables per database
  • • Choose a descriptive, meaningful name

Good Database Name Examples

✅ Good Names

  • • air-quality-2024
  • • water-monitoring
  • • biodiversity-survey
  • • energy-consumption
  • • soil-analysis

❌ Avoid These

  • • my database (spaces)
  • • data@2024 (special chars)
  • • test123 (not descriptive)
  • • Air Quality (uppercase)
  • • db (too short)
